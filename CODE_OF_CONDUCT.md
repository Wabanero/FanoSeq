# Code Of Conduct

FanoSeq discussions should stay technical, careful, and respectful. The project
touches speculative mathematical biology, so criticism is expected and valuable
when it is specific, evidence-based, and aimed at improving the work.

## Expected Behavior

- Be precise about claims, evidence, and uncertainty.
- Treat negative benchmark results as useful information.
- Credit prior work and distinguish mathematical validity from biological
  validation.
- Keep discussions welcoming to contributors with different levels of biology,
  mathematics, and software experience.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Posting private data, credentials, patient information, or non-redistributable
  datasets.
- Misrepresenting FanoSeq as biologically validated when benchmark evidence is
  missing or negative.
- Using the project name or logo in a way that implies permission, endorsement,
  authorship, or collaboration without explicit written consent.

## Enforcement

The maintainer may edit, hide, or remove comments; close issues; decline pull
requests; or restrict participation when behavior conflicts with this code.
