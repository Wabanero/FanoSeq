# Security Policy

## Supported Versions

FanoSeq is pre-release research software. Security fixes target the current
default branch unless a release branch is explicitly announced.

## Reporting A Vulnerability

Report suspected vulnerabilities privately to:

Filippo Bergeretti
fbergeretti@gmail.com

Please include:

- affected commit or version;
- operating system and Python version;
- reproduction steps;
- impact assessment;
- whether any private data or credentials may have been exposed.

Do not open a public issue for credential leaks, arbitrary file access, command
execution, dependency-confusion, or data-exposure reports.

## Data Handling

Do not use this repository with patient-identifiable, controlled-access, or
confidential biological data unless you have an approved local governance
process. The bundled examples and dataset manifests are intended for public,
redistributable source data only.
